Introduction au langage Bash

Qu'est ce que le Bash ?

Le bash est un langage de programmation destiné à l'exécution de commandes. C'est une interface faisant le lien entre le noyau d'un système d'exploitation et un utilisateur, raison pour laquelle ce language est également un shell.
Formelleent, un shell est une couche logicielle, c'est à dire un programme qui permet l'accès aux services du système d'exploitation à un utilisateur ou à d'autres programmes. Sur un système linux ou macintosh, le bash l'interpréteur de commandes utilisé utilisé par défaut.

Commandes courantes

pwd Affiche le chemin d'accès du répertoire courant
cd [chemin] Change le répertoire courant par celui indiqué en argument
ls [chemin] Affiche les informations relatives aux fichiers passés en arguments. Sans arguments, la commande s'applique à l'ensemble des fichiers du répertoire courant.
OPTIONS :
-a/--all s'applique également aux fichiers cachés, c'est à dire dont le nom commence par "."
-l
mkdir [nom_repertoire] Crée un dossier
cp [fichier] [destination] Copie un fichier ou un dossier vers le chemin indiqué
mv [fichier] [destination] Déplace un fichier ou un répertoire vers le chemin inqiqué
rm [fichier] supprime le fichier indiqué
find
grep
chmod [fichier] Rend le fichier exécutable.
cat [fichier] Affiche les premières lignes du fichier
